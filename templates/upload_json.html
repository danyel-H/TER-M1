Créer un évènement avec un fichier JSON





Afin de pouvoir upload l'évènement sur le serveur, veuillez copier coller le JSON ci-dessous et modifier les valeurs qui vous intéressent :

La recurrence peut poser problème si vous ne connaissez pas le format, elle définit quand, combien de fois et jusqu'a où l'évènement doit se répéter.
Voici une aide pour la générer : Lien (Attention, quand vous collez, faites bien à attention à laisser le "RRULE:" au début du champ)

Les dates sont au format ISO-8601, une aide est aussi disponible pour la générer : Lien (Générez votre date et copier le format ISO-8601 en dessous)

Les seuls champs obligatoires sont les champs start et end, vous pouvez supprimer les autres s'ils ne vous intéressent pas 
    {
        "summary": "Le titre",
        "location": "Le lieu",
        "description": "La description",
        "start": {
            "dateTime": "2015-05-28T09:00:00-07:00",
            "timeZone": "Europe/Paris"
        },
        "end": {
            "dateTime": "2015-05-28T17:00:00-07:00",
            "timeZone": "Europe/Paris"
        },
        "recurrence": [
            "RRULE:FREQ=DAILY;COUNT=2"
        ],
        "attendees": [
            {"email": "invite@gmail.com"}
        ]
    }
